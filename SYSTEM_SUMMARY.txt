WHAT IS THIS APPLICATION?
================================================================================
A comprehensive enterprise platform that combines:
  1. Project Management (Atlas)
  2. Activity Monitoring (WorkPulse)
  3. Performance Management (EPR)
  4. Research Collaboration (Labs)

All services work together through a central Orchestrator API Gateway.

USER VALUE PROPOSITION
================================================================================
Organizations get ONE unified platform instead of 4 separate tools:
  ✓ Manage projects and tasks
  ✓ Monitor team productivity in real-time
  ✓ Evaluate employee performance
  ✓ Facilitate research collaboration
  ✓ Get unified dashboards and insights

SYSTEM ARCHITECTURE
================================================================================
                    Orchestrator (Port 9000)
                    ↓
        ┌───────────┼───────────┬───────────┐
        ↓           ↓           ↓           ↓
      Atlas      WorkPulse     EPR        Labs
     (8000)      (8001)      (8003)      (8004)

All services are FastAPI applications running independently.
Orchestrator acts as API Gateway and coordinates between services.

CORE FEATURES
================================================================================

1. PROJECT MANAGEMENT (Atlas - Port 8000)
   - Create and manage projects
   - Break projects into tasks
   - Track issues and bugs
   - Assign work to team members
   - Monitor project progress

2. ACTIVITY MONITORING (WorkPulse - Port 8001)
   - Track user activity throughout the day
   - View today's activity summary
   - Log custom activities
   - Team-wide activity dashboard
   - Productivity statistics

3. PERFORMANCE MANAGEMENT (EPR - Port 8003)
   - Performance scoring system
   - Goal setting and tracking
   - Performance reviews
   - Peer feedback collection
   - Skill assessment
   - Team performance metrics

4. RESEARCH COLLABORATION (Labs - Port 8004)
   - Lab management
   - Researcher profiles
   - Collaboration suggestions
   - Automated collaboration emails

API ENDPOINTS SUMMARY
================================================================================

AUTHENTICATION (/api/v1/auth)
  POST   /register              - Create new user
  POST   /login                 - Login user
  GET    /me                    - Get current user
  POST   /organizations         - Create organization
  GET    /organizations/{id}    - Get organization

PROJECTS (/api/v1/projects)
  GET    /                      - Get user projects
  GET    /{id}/tasks            - Get project tasks
  GET    /{id}/issues           - Get project issues
  POST   /tasks                 - Create task

MONITORING (/api/v1/monitoring)
  GET    /activity/{user_id}    - Get user activity
  GET    /activity/{user_id}/today - Get today's activity
  POST   /activity/log          - Log activity
  GET    /team/{org_id}         - Get team activity
  GET    /stats/{user_id}/productivity - Get productivity stats

PERFORMANCE (/api/v1/performance)
  GET    /user/{id}/score       - Get performance score
  GET    /user/{id}/goals       - Get user goals
  POST   /goals                 - Create goal
  GET    /user/{id}/reviews     - Get reviews
  GET    /user/{id}/feedback    - Get feedback
  GET    /user/{id}/skills      - Get skills
  GET    /team/{org_id}/performance - Get team performance

RESEARCH (/api/v1/research)
  GET    /labs                  - Get all labs
  GET    /labs/{id}             - Get specific lab
  POST   /labs                  - Create lab
  GET    /researchers           - Get all researchers
  GET    /labs/{id}/researchers - Get lab researchers
  POST   /researchers           - Create researcher
  GET    /collaborations        - Get collaboration suggestions
  POST   /collaborations/email  - Generate collaboration email

SYSTEM (/api/v1)
  GET    /health                - Check system health
  GET    /services              - List all services
  GET    /dashboard             - Get unified dashboard

HOW TO USE
================================================================================

1. START THE SYSTEM
   cd services/orchestrator
   python main.py
   
   This automatically starts all 4 services!

2. REGISTER A USER
   curl -X POST http://localhost:9000/api/v1/auth/register \
     -H "Content-Type: application/json" \
     -d '{
       "email": "user@example.com",
       "name": "John Doe",
       "password": "password123",
       "role": "member"
     }'

3. LOGIN
   curl -X POST http://localhost:9000/api/v1/auth/login \
     -H "Content-Type: application/json" \
     -d '{
       "email": "user@example.com",
       "password": "password123"
     }'

4. USE THE API
   curl -X GET http://localhost:9000/api/v1/projects/ \
     -H "Authorization: Bearer YOUR_TOKEN"

5. VIEW DOCUMENTATION
   - Swagger UI: http://localhost:9000/docs
   - ReDoc: http://localhost:9000/redoc
